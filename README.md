# FoodNow-webApp
full stack MERN web application that incorporates a plethora of features reminiscent of renowned food delivery websites and apps.
tech used : 
bootstrap, mongoose, react, express, bcryptjs, jsonwebtoken

-----COMPONENTS---- components like footer , navbar, cards which are used multiple times are stored in component folder
-----SCREENS------- screens folder stores important pages like home.js, login.js, signup.js, myorder.js
